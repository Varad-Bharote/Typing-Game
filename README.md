# Typing Speed Game

A fun and interactive typing game to improve your typing speed and accuracy. Test your skills against the clock with words from comprehensive dictionaries including Oxford, Cambridge, and more!

## Features

- **Comprehensive Dictionary**: Over 200 words from various categories including everyday language, technology, science, music, sports, and challenging vocabulary
- **Customizable Time Limits**: Choose from 30, 60, 120, or 300 seconds
- **Real-time Statistics**: Track your score, Words Per Minute (WPM), and accuracy percentage
- **High Score Tracking**: Saves your best score using browser localStorage
- **Sound Effects**: Audio feedback for correct and incorrect typing
- **Responsive Design**: Works on desktop and mobile devices
- **Help Section**: Built-in instructions for new players

## How to Play

1. Select your preferred time limit
2. Click "Start Game" to begin
3. Type the displayed word exactly as shown
4. Submit by typing the full word correctly or pressing Enter/Space
5. Try to achieve the highest score possible before time runs out!

## Word Categories

The game includes words from these categories:
- Basic vocabulary
- Common nouns, verbs, and adjectives
- Animals and nature
- Food and drinks
- Technology and computing
- Science and mathematics
- Music and instruments
- Sports and activities
- Challenging and complex words

1. Download or clone this repository
2. Open `index.html` in your web browser
3. Start typing!

## Deploying to GitHub

This repository includes helper scripts to easily set up Git and deploy to GitHub:

1. **Install Git**: Download and install Git from [https://git-scm.com/downloads](https://git-scm.com/downloads)

2. **Configure Git**: Open a terminal and set your identity:
   ```bash
   git config --global user.email "you@example.com"
   git config --global user.name "Your Name"
   ```

3. **Initialize Repository**: Run the setup script:
   ```bash
   setup_git_repository.bat
   ```

4. **Create GitHub Repository**: 
   - Go to [https://github.com/new](https://github.com/new)
   - Name your repository (e.g., "Typing-Game")
   - Do NOT initialize with a README

5. **Push to GitHub**: 
   - Edit `push_to_github.bat` to replace `your-username` and `your-repo-name` with your actual GitHub username and repository name
   - Run the push script:
   ```bash
   push_to_github.bat
   ```
